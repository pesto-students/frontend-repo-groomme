# Groomme

## Setup Configuration

### Prerequisites

Make sure you have the following installed:

- Node.js
- npm (Node Package Manager)

Navigate to the backend directory:

```bash
cd Backend
npm install
npm run dev
```

Navigate to the frontend directory:

```bash
cd Frontend
npm install
npm run dev
```
